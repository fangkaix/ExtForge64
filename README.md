# ExtForge64

ExtForge64 是一个用于在 64 位平台上创建 ext4 文件镜像的脚本工具。默认3G大小

## 功能

- **创建 ext4 文件镜像**：在用户指定的路径下创建 ext4 文件系统的镜像。
- **适用于 64 位平台**：专为 64 位系统设计，确保镜像文件在该平台上正常工作。

## 文件结构
```bash
ExtForge64/
├── aarch64
│ └── make_ext4fs
├── generate_image.sh
├── x86_64
  └── make_ext4fs
```
## 使用方法

### 前提条件

- `make_ext4fs` 已包含在项目中，并且位于相应的架构目录中 (`aarch64` 或 `x86_64`)。
- 脚本应具有执行权限。

### 步骤

1. **准备工作**：
   - 将需要放入镜像文件的内容放入名为 `user_app` 的文件夹中。

2. **运行脚本**：
   - 打开终端，导航到脚本所在的目录。
   - 执行以下命令：
```bash
./generate_image.sh
```
3. **按提示操作**：
- 脚本将引导你输入镜像文件的保存路径，确认后即可开始创建镜像文件。

4. **等待完成**：
- 脚本将自动创建 ext4 格式的镜像文件，并在指定路径下生成镜像文件。

## 注意事项

- 确保目标文件夹具有写权限，以便在其中创建镜像文件。
- 使用时请谨慎，特别是在指定保存路径时，请确保磁盘空间充足且路径有效。
